This project is maintained by Pablo Cingolani (pcingola@users.sourceforge.net)

Please send any bug reports to pcingola at users.sourceforge.net. A bug report should include: 

	- An explanation of the bug condition

	- A minimal bug-demonstrating test case 
        (i.e. include FCL file and java code needed to run the test case)

	- The conditions for the bug to occur (e.g. input variables)

	- jFuzzyLogic version you are using
